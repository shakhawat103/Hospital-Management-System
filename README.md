# Hospital Management System

A role-based Hospital Management System built with ASP.NET Core MVC and deployed on Microsoft Azure.  
It provides secure authentication and appointment management for hospital operations.

---

## Live Demo

https://hospital-mvc-app-d5gjdvgpgsa3czdk.southeastasia-01.azurewebsites.net

---

## Tech Stack

- ASP.NET Core MVC  
- C#  
- Entity Framework Core  
- LINQ  
- SQL Server (Azure SQL Database)  
- ASP.NET Identity  
- AutoMapper  
- Repository Pattern & Unit of Work  

---

## Key Features

### Admin
- Manage patients, doctors, appointments, and users

### Patient
- View doctors  
- Book appointments  
- Update profile  

---

## Deployment

- Microsoft Azure App Service  
- Azure SQL Database integration

## Concepts Used
